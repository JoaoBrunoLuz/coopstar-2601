# Coopstar Express - Landing Page

Modernização da landing page da Coopstar Express utilizando React, TypeScript e Tailwind CSS.

## 🚀 Tecnologias

Este projeto foi desenvolvido com as seguintes tecnologias:

- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [Lucide React](https://lucide.dev/)

## 💻 Projeto

A Coopstar Express é especialista em serviços de entregas e coletas com mais de 9 anos de experiência no mercado. Esta landing page foi refatorada para oferecer uma experiência moderna, rápida e responsiva.

## 🛠️ Instalação e Execução

1. Clone o repositório:
   ```bash
   git clone https://github.com/JoaoBrunoLuz/coopstar-2601.git
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
